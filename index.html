Introduction

The data for this project come from this source: http://groupware.les.inf.puc-rio.br/har. The project consists of building a machine learning algorithm to predict activity quality from activity monitors.

Getting data

The data is read here and training and testing set are created

library(caret)
library(rattle)
library(randomForest)

alldata = read.csv("pml-training.csv")
set.seed(32323)

inTrain <- createDataPartition(y=alldata$classe,p=0.7, list=FALSE) 
training <- alldata[inTrain,]
testing <- alldata[-inTrain,]
